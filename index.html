Współbieżność w Java z wykorzystaniem “concurrency primitives”

Jesteś programistą - super! Wątki i współbieżność budzą w Tobie strach? Mam dla Ciebie szkolenie, które na zawsze pozwoli Ci się pozbyć takich obaw i lęków. Podczas tego szkolenia poznasz bliżej “concurrency primitives” dostępne w Java SDK, czyli pakiet java.util.concurrent. Nauczysz się jak z podstawowych elementów, takich jak: kolejki, blokady, czy bariery budować czytelne rozwiązania, jak analizować ich wydajność i jak je testować

Współbieżność to nasza programistyczna codzienność. Czas ją ujarzmić!

Gdy pojawia się potrzeba wykorzystania mocy wielu procesorów, sięgamy po rozwiązania operujące na wyższym poziomie abstrakcji, czyż nie? Sięgamy po Akka, RxJava, Reactor. Czy tak musi być? Czy Java nie dostarcza mechanizmów pracy z wątkami i tworzenia wydajnego i czytelnego kodu współbieżnego? Podczas tego szkolenia odpowiem na te pytania i inneh pytania. Zdobędziesz niezbędną wiedzę z zakresu tworzenia bezpiecznego, współbieżnego kodu.
  • Wprowadzenie do współbieżności
  • Java Memory Model
  • Kolejki, bariery, wątki, mechanizmy synchronizacji
  • Testowanie i poprawność naszego kodu
